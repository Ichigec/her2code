# 27-Point Validation Checklist for Motivation Letters

Run this checklist before finalizing any motivation letter. All 27 must pass.

## Structure (6 points)

| # | Criterion |
|---|---|
| 1 | Opens with PROJECT (not biography, not name, not age) |
| 2 | Project described concretely (technologies, features, deployment surfaces) |
| 3 | 2-3 measurable achievements from resume (%, X, team size) |
| 4 | WHY THIS PROGRAM section names specific details (leader, pivot, award, stats) |
| 5 | Career goal is specific role, not "I want to develop" |
| 6 | Closing ties back to the program's value proposition |

## Program knowledge (8 points)

| # | Criterion |
|---|---|
| 7 | Program named explicitly (AI Talent Hub, not just "ИТМО") |
| 8 | Admission format named (JMLC, portfolio contest, or exams) |
| 9 | Program leader referenced by name when possible |
| 10 | Recent program pivot/changes cited (must be from current year) |
| 11 | Awards/recognition mentioned (Generation AI, Yandex ML Prize, etc.) |
| 12 | Competition stats cited (X applicants per place, Y budget spots) |
| 13 | Partner companies named (X5, МТС, Sber AI, etc.) |
| 14 | Specific tracks/roles mentioned (Agentic AI, ML Engineer, etc.) |

## Candidate differentiators (5 points)

| # | Criterion |
|---|---|
| 15 | Unique angle identified (non-standard background as asset) |
| 16 | Teaching/mentoring experience mentioned if exists |
| 17 | Industry expertise framed as PROGRAM asset, not personal deficiency |
| 18 | Technologies listed match program's tracks |
| 19 | Any potential weaknesses reframed as strengths |

## Tone and authenticity (5 points)

| # | Criterion |
|---|---|
| 20 | Confident, not apologetic ("I come to strengthen, not to learn from zero") |
| 21 | Concrete, not vague (every claim backed by number or name) |
| 22 | No flattery without evidence |
| 23 | Consistent register (formal "вы" throughout) |
| 24 | Reads as one person's authentic voice, not committee-written |

## Technical constraints (3 points)

| # | Criterion |
|---|---|
| 25 | ≤2500 characters with spaces |
| 26 | Zero anti-pattern phrases detected |
| 27 | Fits 1 A4 page when rendered (verified via char count) |

## Anti-pattern phrases (must be ABSENT)

If any of these appear, the letter fails criterion 26:

- "хочу развиваться"
- "мечтаю"
- "с детства"
- "всегда интересовался"
- "очень нравится"
- "пожалуйста"
- "буду благодарен"
- "надеюсь"
- "прошу рассмотреть"
- "заранее спасибо"
- "ваша программа лучшая"
- "я уверен, что"
- "с большим интересом"
- "искренне"
