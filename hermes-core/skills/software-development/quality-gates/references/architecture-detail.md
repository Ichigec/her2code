# Quality Gates — Enforcement-First Architecture (detail)

> Reference for quality-gates skill. Full architecture from Architect agent.

## GatePassport — Cryptographic Certificate

Generated by `quality_gate_runner.py` when ALL 7 mandatory gates pass:

```json
{
  "cycle_id": "login-feature_20260621_120000",
  "timestamp": 1718985637.123,
  "verdict": "ALL_PASSED",
  "total_gates": 7,
  "passed_gates": 7,
  "git_commit": "abc123def456",
  "workdir_hash": "sha256:def456...",
  "hmac_signature": "abcdef0123456789..."
}
```

- Secret key: `~/.hermes/gates/.gate-secret` (0600)
- 30-minute TTL
- Bound to git commit + workdir hash — cannot be reused across commits
- Verified by VPS `pre-deploy-gate.sh` before accepting deploy

## Gate Dependency DAG

```
Level 0 (alone):         BusinessAnalysisGate     ← ROOT, mandatory, always first
Level 1 (parallel):      BuildGate, SecurityGate, TestGate, CoverageGate, IntegrationGate
Level 2 (after build):   AcceptanceGate
```

BA Gate runs alone at Level 0. If it fails (traceability gaps), remaining 6 gates skip entirely — fast-fail.

## Modes — Depth, Not Disable

Modes NEVER disable gates (except deployment-gate). They change CHECK DEPTH:

| Mode | Build | Test | Security | Coverage |
|------|-------|------|----------|----------|
| speed | Incremental | Affected tests | bandit only | Changed files only |
| balanced | Full | Full | bandit+semgrep+gitleaks | ≥80% |
| quality | Full | Full ×3 (flaky) | All 4 tools | ≥90%, per-file ≥70% |

## Gate History DB

`~/.hermes/gate_history.db` (SQLite) — records every gate run:
- `gate_runs` — verdict, score, diagnostic per iteration
- `gate_checks` — per-check results with actual vs expected
- `detect_stuck()` — 3 consecutive same diagnostic → stuck warning
- `detect_regression()` — score drop from previous iteration
- `get_cycle_metrics()` — aggregate stats for Auditor

## Tamper-Evidence

| Bypass attempt | Detection | Result |
|---------------|-----------|--------|
| `git commit --no-verify` | commit-msg hook still runs | BLOCKED |
| Forged GatePassport | HMAC-SHA256 verification | REJECTED |
| `git push --force` | pre-push hook | BLOCKED |
| Manual SSH deploy | VPS pre-deploy-gate.sh | BLOCKED |
| Delete git hooks | enforce-workspace.py protects | DETECTED |
| Modify gate runner | Self-integrity SHA256 check | EXIT code 3 |
| Disable gates in config | MANDATORY_GATES in registry | IGNORED |
