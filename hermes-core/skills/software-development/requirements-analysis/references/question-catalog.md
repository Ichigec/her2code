# Business Analysis Question Catalog

~40 structured interview questions for business analysts, organized by category.
Based on BABOK® v3 (Strategy Analysis, Elicitation & Collaboration, Requirements Analysis),
Palomares et al. (2021) — "State-of-Practice in Requirements Elicitation",
and IEEE RE 2021 — "Typology of Questions for RE Interviews".

## Category A: Context & Strategy (7 questions)

| # | Question | Purpose |
|---|----------|---------|
| A1 | What is the main goal of your department/project? | Strategic context |
| A2 | What KPIs/metrics measure success? | Success criteria |
| A3 | Who are the primary customers/users of this process? | Target audience |
| A4 | How does this initiative connect to company strategy? | Strategic alignment |
| A5 | What external factors (regulators, market) affect the process? | Context constraints |
| A6 | What happens if nothing changes? | Justification for change |
| A7 | What adjacent systems/processes does the project touch? | Project boundaries |

## Interview Frameworks

| Framework | Structure | Best for |
|-----------|----------|----------|
| **SPIN** | Situation → Problem → Implication → Need-Payoff | Moving from surface wants to deep needs |
| **GROW** | Goal → Reality → Options → Will | Structuring workshops |
| **5 Whys** | 5× "Why?" → root cause | Root cause analysis before requirements |
| **Socratic** | 6 types: clarify, assumptions, evidence, perspectives, consequences, meta | Uncovering implicit requirements |
| **MoSCoW** | Must / Should / Could / Won't | Prioritization in consolidation phase |

## Interview Protocol (BABOK-aligned)

```
PHASE 1 — PREPARATION
├── Define interview goals
├── Identify stakeholders (Stakeholder Map)
├── Prepare interview guide
└── Agree on format and duration

PHASE 2 — OPENING
├── Introduce yourself, explain purpose
├── State confidentiality
├── Request recording permission
└── Set timing expectations

PHASE 3 — BODY (Funnel approach)
├── Open questions (broad context)
├── Probing questions (narrowing focus)
├── Closed questions (confirming details)
└── Summarizing (checking understanding)

PHASE 4 — CLOSING
├── Summarize key findings
├── Clarify unclear points
├── Agree on next steps
└── Thank the stakeholder
```

Source: BABOK v3, Palomares et al. (2021), IEEE RE 2021, Interview Patterns for RE (Scheinholtz & Wilmont, 2011).
