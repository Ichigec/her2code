Агентный пайплайн
на базе Hermes

Эволюция персонального AI-агента

Идея

Сложный пайплайн из дешёвых моделей
даёт качество дорогих

📊 Больше токенов
💰 Но дешевле за токен
🎯 Качество = архитектура пайплайна

Модели

🏠 Локальные: Qwen3.6-35B, DiffusionGemma, Nex-N2

☁️ DeepSeek: дешёвые облачные токены

Это третий шаг эволюции персонального агента:
одиночный агент → оркестратор → пайплайн-фабрика

Суть: не «один дорогой агент», а конвейер из дешёвых — получил задачу, разложил по фазам, исполнил параллельно, проверил гейтами, сохранил опыт

Экономика: дешёвые токены × сложный пайплайн

Одиночный агент

$15 / 1M токенов

Claude Opus

1 проход

Без верификации

Без памяти

→

Пайплайн из дешёвых

$0.27 / 1M токенов

DeepSeek V4

5+ проходов (анализ → код → ревью → правки → аудит)

Гейты на каждой фазе

Память + навыки

55× дешевле за токен. Можно потратить в 20× больше токенов на верификацию — и всё равно выйти дешевле, но с качеством выше за счёт многофазной проверки.

Как достигается качество

🎯 Требования → формализация задачи

🔍 Анализ → 5 Whys, альтернативы

📚 Research → поиск лучших практик

🏗️ Архитектура → контракты модулей

💻 Разработка → 5 стилей × N параллельно

🔒 Security → SAST (bandit, gitleaks, semgrep)

✅ Приёмка → тесты + traceability matrix

Итоговая арифметика

Claude: 10K токенов × $15 = $0.15

Пайплайн: 200K токенов × $0.27 = $0.05

+ качество выше за счёт:

• Многофазной верификации

• Параллельной разработки (выбор лучшего)

• Накопления навыков (не повторяет ошибки)
