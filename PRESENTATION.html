Агентный пайплайн на базе Hermes

Третий шаг эволюции персонального AI-агента
одиночный агент → оркестратор → пайплайн-фабрика

Почему пайплайн, а не один агент

Один агент = одна модель = один проход.
Результат: нет верификации, нет памяти, ошибки повторяются.

Пайплайн = 10 фаз × дешёвые токены (локальные Qwen3.6 / DeepSeek $0.27/1M):

🎯 Требования → 🔍 Анализ → 📚 Research → 🏗️ Архитектура

→ 📋 План → ⚡ Pre-Flight → 💻 Dev ×N → 🔒 SAST → 🚀 Деплой → ✅ Приёмка

Каждая фаза = отдельный вызов модели со своей специализацией.
Больше токенов, но они в 55× дешевле → итоговая стоимость ниже, качество выше.

Эволюция: v1 — один агент (2025) → v2 — оркестратор (нач. 2026) → v4 — пайплайн (июль 2026) → v5+ — фабрика агентов
